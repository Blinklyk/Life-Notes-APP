# 随心记

该上下文描述用户如何留下生活片段、按天整理它们并回望自己的经历。这里统一产品语言，不包含具体实现方式。

## Language

**随心记录**：
用户在某一时刻主动保存的一条原始生活片段，可包含文字、图片、视频或语音中的一种或多种内容。
_Avoid_: 动态、帖子、日记

**记录日**：
按用户当时的本地日期划分的自然日，是聚合随心记录、每日感受和随心日记的边界。
_Avoid_: 文件夹、会话

**每日素材**：
属于同一记录日的全部随心记录的集合，是生成随心日记时使用的事实来源。
_Avoid_: 日记、总结

**随心日记**：
系统根据某个记录日的每日素材生成、并允许用户编辑的一篇独立文本；它不会替代或修改原始随心记录。
_Avoid_: 原始记录、AI 回复

**表达风格**：
用户希望随心日记采用的叙述口吻与表达偏好，可由用户明确设置，并在获得许可后从其编辑行为中逐步调整。
_Avoid_: 人格、用户画像

**每日感受**：
用户对一个记录日的主观情绪评价；它表达当天的总体感受，不等同于医学或心理健康判断。
_Avoid_: 心理诊断、情绪检测

**重要日**：
被用户主动标记为值得特别回看的记录日；是否重要与当天素材数量无关。
_Avoid_: 收藏记录、系统推荐
